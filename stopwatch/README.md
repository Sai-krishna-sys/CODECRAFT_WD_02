# Stopwatch Web Application

A simple stopwatch web application built with HTML, CSS, and JavaScript.

## Features

- Start, pause, and reset the stopwatch
- Record lap times
- Accurate timing with millisecond precision
- Clean, responsive design

## Usage

1. Open `index.html` in a web browser
2. Click "Start" to begin timing
3. Click "Pause" to pause the timer
4. Click "Reset" to reset the timer and clear lap times
5. Click "Lap" to record a lap time while running

## Files

- `index.html`: Main HTML structure
- `styles.css`: CSS styling
- `script.js`: JavaScript functionality

## Browser Compatibility

Works in all modern web browsers that support JavaScript.
